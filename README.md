# cybersecurity-home-lab
Beginner cybersecurity lab using Kali Linux and Metasploitable

## Objective
To build a safe penetration testing lab using Kali Linux and Metasploitable to practice basic cybersecurity concepts.

## Lab Setup
- VirtualBox
- Kali Linux
- Metasploitable 2
- Host-Only Network Configuration
